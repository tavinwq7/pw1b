Tags que não são auto fechantes


HEAD

A tag HEAD não exibe conteúdo visível na página, mas contém informações cruciais para o funcionamento, a aparência e a estruturação do site, como estilos, scripts, metadados, título da página, etc.

BODY

A tag é uma tag não auto fechante e representa o corpo do documento HTML. Tudo que é visível para o usuário em uma página web deve estar dentro dessa tag. A tag BODY só pode existir uma vez dentro de um documento HTML. Ela deve ser usada dentro da estrutura padrão do HTML, depois da HEAD.
